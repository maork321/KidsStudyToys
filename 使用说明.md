
# 儿童教育平台 - 使用说明

## 项目简介
专为4-8岁儿童设计的多学科在线学习平台，包含语文、数学、英语、科学等科目，支持离线本地运行，无需配置复杂服务器。

## 功能特点
- ✅ 多学科课程体系
- ✅ 互动式学习模块
- ✅ 学习进度追踪
- ✅ 用户注册登录
- ✅ 个性化路径推荐
- ✅ 成就激励系统（积分商城）
- ✅ 全设备自适应（手机/平板/TV/PC）
- ✅ Docker容器化部署，支持多架构
- ✅ 离线本地运行

## 快速开始

### 方式一：直接开发运行
```bash
# 安装依赖
npm install

# 开发模式运行
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

### 方式二：Docker部署（推荐）
```bash
# 构建并启动容器
docker-compose up -d --build

# 或使用 docker buildx 多架构构建
docker buildx build --platform linux/amd64,linux/arm64,linux/arm/v7 -t kids-edu-platform:latest --load .

# 启动后访问
http://localhost:8080
```

## 技术架构
- 前端框架：React 18 + TypeScript
- UI样式：Tailwind CSS 3
- 状态管理：Zustand
- 路由：React Router DOM 7
- 构建工具：Vite
- 本地存储：LocalStorage + Zustand Persist
- Web服务器：Nginx（Docker容器内）

## 项目结构
```
/workspace
├── src/
│   ├── pages/              # 页面组件
│   ├── components/         # 公共组件
│   ├── store/              # 状态管理
│   ├── types/              # 类型定义
│   ├── data/               # 初始数据
│   ├── App.tsx             # 应用入口
│   └── main.tsx            # 根组件
├── public/                 # 静态资源
├── dist/                   # 构建输出

├── docker-compose.yml      # Docker Compose配置
└── nginx.conf              # Nginx配置
```

## 数据存储
- 所有数据存储在浏览器本地存储中
- 支持多用户账户切换
- 数据持久化，刷新页面不会丢失

## 支持平台
- ✅ Linux/amd64
- ✅ Linux/arm64 (树莓派4/5等)
- ✅ Linux/arm/v7 (树莓派3等)
- ✅ Windows/macOS (通过Docker Desktop)

## 浏览器兼容性
- Chrome/Edge (推荐)
- Safari
- Firefox
- 需要支持ES6+的现代浏览器

## 开发说明
- 添加新课程：编辑 `src/data/initialData.ts`
- 自定义主题：修改颜色方案在各页面中
- 添加新功能：在 `src/pages/` 和 `src/components/` 中开发

## 注意事项
- 数据仅存储在本地浏览器中，清除浏览器数据会丢失
- 建议定期备份localStorage数据
- 适合家庭或小型机构使用
