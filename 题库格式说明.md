## 通用格式

所有题库都遵循以下基本结构：

```json
{
  "id": "math-bank-l1",
  "subjectId": "math",
  "name": "数学入门题库",
  "description": "适合4-5岁小朋友的数学题目",
  "questions": [
    {
      "id": "q1",
      "type": "multiple_choice",
      "difficulty": 1,
      "content": "1 + 2 = ?",
      "options": ["1", "2", "3", "4"],
      "correctAnswer": "3",
      "explanation": "1个苹果加2个苹果等于3个苹果",
      "image": "https://example.com/image.png"
    }
  ],
  "createdAt": "2024-01-01T00:00:00.000Z",
  "updatedAt": "2024-01-01T00:00:00.000Z"
}
```

### 字段说明
- **id**: 题库唯一标识
- **subjectId**: 所属学科 (chinese/math/english/science)
- **name**: 题库名称
- **description**: 题库描述
- **questions**: 题目数组
- **createdAt/updatedAt**: 创建/更新时间

### 题目类型
- `multiple_choice`: 选择题
- `true_false`: 判断题
- `fill_blank`: 填空题
- `matching`: 连线题
- `calculation`: 计算题

### 难度等级
- `1`: 简单（适合3-4岁）
- `2`: 中等（适合5-6岁）
- `3`: 困难（适合7-8岁）

```json
{
  "id": "chinese-bank-l1",

      "correctAnswer": "一",
      "explanation": "这是数字'一'的汉字写法",
      "image": "https://example.com/yi.png",
      "pinyin": "yī",
      "radicals": ["一"],
      "strokes": 1
    },
    {
      "id": "c2",
      "type": "pinyin",
      "difficulty": 1,
      "content": "这个拼音怎么读？",
      "options": ["a", "o", "e"],
      "correctAnswer": "a",
      "explanation": "a是汉语拼音的第一个韵母"

    {
      "id": "c3",
      "type": "poem",

      "correctAnswer": "曲项向天歌",
      "explanation": "《咏鹅》唐·骆宾王：鹅鹅鹅，曲项向天歌..."
    },
    {
      "id": "c4",
      "type": "reading",
      "difficulty": 3,
      "content": "看图讲故事：图片中发生了什么？",
      "options": ["小朋友在读书", "小朋友在吃饭", "小朋友在睡觉"],
```

### 语文题目类型
- `character`: 汉字认知
- `pinyin`: 拼音学习
- `poem`: 古诗学习
- `reading`: 阅读理解

### 语文专有字段
- **pinyin**: 拼音
- **radicals**: 部首
- **strokes**: 笔画数

```json
{
  "id": "math-bank-l1",
  "subjectId": "math",
  "name": "数学入门题库",
  "description": "认识数字、学习计算",
  "questions": [
    {
      "id": "m1",
      "type": "calculation",
      "difficulty": 1,
      "content": "1 + 2 = ?",
      "options": ["2", "3", "4"],
      "correctAnswer": "3",
      "explanation": "1个苹果加2个苹果等于3个苹果",
      "operand1": 1,
      "operand2": 2,
      "operator": "+",
      "unit": ""
    },
    {
      "id": "m2",
      "type": "calculation",
      "difficulty": 1,
      "content": "5 - 3 = ?",
      "options": ["1", "2", "3"],
      "correctAnswer": "2",
      "explanation": "5个苹果吃掉3个，还剩2个",
      "operand1": 5,
      "operand2": 3,
      "operator": "-"
    },
    {
      "id": "m3",
      "type": "multiple_choice",
      "difficulty": 1,
      "content": "🐱🐱🐱 有几只小猫？",
      "options": ["2只", "3只", "4只"],
      "correctAnswer": "3只",
      "explanation": "数一数，一共有3只小猫"
    },
    {
      "id": "m4",
      "type": "geometry",
      "difficulty": 2,
      "content": "这是什么形状？",
      "options": ["圆形", "三角形", "正方形"],
      "correctAnswer": "圆形",
      "explanation": "圆形的特点是边缘圆滑"
    },
    {
      "id": "m5",
      "type": "application",
      "difficulty": 3,
      "content": "小明有3个苹果，妈妈又给了他2个，现在小明有几个苹果？",
      "options": ["3个", "4个", "5个"],
      "correctAnswer": "5个",
      "explanation": "3 + 2 = 5",
      "unit": "个"
    }
  ],
  "createdAt": "2024-01-01T00:00:00.000Z",
  "updatedAt": "2024-01-01T00:00:00.000Z"
}
```

### 数学题目类型
- `calculation`: 计算题目
- `geometry`: 几何认知
- `application`: 应用题目

### 数学专有字段
- **operand1**: 操作数1
- **operand2**: 操作数2
- **operator**: 操作符 (+/-/×/÷)
- **unit**: 单位

## 上传和管理

将题库JSON文件放入 `src/data/question-banks/` 目录下，在应用中引用即可。

### 命名规范
- 文件名：`{subject}-{level}-{bank-number}.json`
- 示例：`math-l1-1.json`, `chinese-l2-2.json`

### 批量导入
多个题库文件可以同时放在同一个目录下，应用会自动加载。
